About

The Golden Plains Shire Project aims to highlight the Shire's history through an interactive web museum. The shire doesn’t currently have any platforms where users can find in-depth historical information about the region. We want to create a web museum which will allows users to gain knowledge and discover a new perspective of the shire. We believe having this kind of information available to the public will help people develop an understanding and better connect to the land. Our team’s goal for this project is the create an intuitive interactive website which will engage users and help them learn about the history.
